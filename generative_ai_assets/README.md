# Generative AI Images

This folder is intended for source assets created with generative AI (my examples are generated with Chat GPT Images 1.5 and 2).

Use it for individual symbols, patterns, or intermediate PNG outputs before they are turned into sprite bundles.

Recommended use:
- keep raw symbol images here (you can use the png_cutter if nescesarry to cut them, make them seamless or to remove a white background)
- keep raw pattern images here
- keep experiment outputs here before importing them into the Sprite Builder

If you want to share finished sprites, use [`shared_sprites`](../shared_sprites) instead.
