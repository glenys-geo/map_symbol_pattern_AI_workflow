# Generative AI Assets

This folder is intended for source assets created with generative AI.

Use it for individual symbols, patterns, or intermediate PNG outputs before they are turned into sprite bundles.

Recommended use:

- keep raw symbol images here
- keep raw pattern images here
- keep experiment outputs here before importing them into the Sprite Builder

If you want to share finished sprite bundles, use [`shared_sprites`](../shared_sprites) instead.
