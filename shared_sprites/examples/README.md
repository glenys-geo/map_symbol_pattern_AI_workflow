# Example Sprite Bundles

This folder is intended for curated example bundles that are shared with the repository.

Recommended structure per example set:

- `example-name/sprite.png`
- `example-name/sprite.json`
- optional `example-name/sprite@2x.png`
- optional `example-name/sprite@2x.json`
- optional `example-name/notes.md`

Keep each example bundle in its own subfolder so the files stay grouped together.
