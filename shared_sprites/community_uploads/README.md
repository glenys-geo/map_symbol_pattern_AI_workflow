# Community Uploads

This folder is intended for sprite bundles that users want to share publicly in the repository.

Recommended structure per upload:

- `author_or_project_name/sprite.png`
- `author_or_project_name/sprite.json`
- optional `author_or_project_name/sprite@2x.png`
- optional `author_or_project_name/sprite@2x.json`
- optional `author_or_project_name/notes.md`

Please keep every shared bundle in its own subfolder.
